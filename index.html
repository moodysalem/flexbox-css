flexbox-css

flexbox-css is a utility stylesheet with responsive CSS classes representing all the configurations of flexbox, so you don't have to worry about vendor prefixing and you don't have to mix in flexbox with your own styles. You just apply the utility classes you need to get the proper layout for different screen sizes.

Install

npm i --save-dev flexbox-css

Development

The build process for this project is sass to autoprefixer to cleancss. Run npm run build to make your own build. When doing development, use npm run dev to watch the source files for changes and automatically rebuild.

Usage

If you are using webpack to include CSS with a style loader, then it's as simple as

import 'flexbox-css';

Otherwise, you can download flexbox-css.css and use it like any other stylesheet

Child Classes

Customized Grow/Shrink

0 through 12 are supported for .flex-grow and .flex-shrink classes

.item.flex-grow-5
.item.flex-grow-2
.item.flex-shrink-2

Ordering

0 through 12 are supported for the order classes and can be used to reorder the children

.item.order-3 1
.item.order-1 2
.item.order-2 3

Responsive Classes

All the classes mentioned above also have responsive variants of xs, sm, md, lg and xl. The default behavior is to apply to screens of all sizes. Applying a prefix to the class name will limit the class to only apply to that screen size range

Prefix Screen Sizes
No Prefix xs- sm- sm-up- sm-down- md- md-up- md-down- lg- lg-up- lg-down- xl-
Min All 0 480px 480px 0 768px 768px 0 992px 992px 0 1200px
Max All 480px 768px up 768px 992px up 992px 1200px up 1200px up
