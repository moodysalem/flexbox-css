Flexboxy

Flexboxy is a simple utility stylesheet with responsive CSS classes representing all the common configurations of flexbox, so you don't have to worry about vendor prefixing and you don't have to mix in flexbox with your own styles.

Install

npm i --save-dev flexboxy

Development

The build process for this project is sass to autoprefixer to cleancss. Run npm run build to make your own build. When doing development, use npm run dev to watch the source files for changes and automatically rebuild.

Usage

If you are using webpack to include CSS with a style loader, then it's as simple as

import 'flexboxy';

Otherwise, you can download flexboxy.css and use it like any other stylesheet

Parent Classes

Direction

Row and Column directions are supported equally. The parent must explicitly describe its orientation

.flex.row > .item
.flex.row > .item
.flex.row > .item
.flex.col > .item
.flex.col > .item
.flex.col > .item

Reverse

Add the reverse class to your .flex.row or .flex.col to swap the order

.flex.row.reverse > .item 1
.flex.row.reverse > .item 2
.flex.row.reverse > .item 3
.flex.col.reverse > .item 1
.flex.col.reverse > .item 2
.flex.col.reverse > .item 3

Wrapping

Add .nowrap, .wrap or .wrap-reverse to your .flex.row or .flex.col to change wrapping behavior

.flex.row.wrap > .item 1
.flex.row.wrap > .item 2
.flex.row.wrap > .item 3
.flex.row.wrap > .item 4
.flex.row.wrap > .item 5
.flex.row.wrap > .item 6
.flex.col.wrap-reverse > .item 1
.flex.col.wrap-reverse > .item 2
.flex.col.wrap-reverse > .item 3
.flex.col.wrap-reverse > .item 4
.flex.col.wrap-reverse > .item 5
.flex.col.wrap-reverse > .item 6

Justification

Add .justify-(start/end/center/space-between/space-around) to .flex.row or .flex.col modify justification behavior.

.flex.row.justify-start > .item 1
.flex.row.justify-start > .item 2
.flex.row.justify-start > .item 3
.flex.row.justify-center > .item 1
.flex.row.justify-center > .item 2
.flex.row.justify-center > .item 3
.flex.row.justify-end > .item 1
.flex.row.justify-end > .item 2
.flex.row.justify-end > .item 3
.flex.row.justify-space-between > .item 1
.flex.row.justify-space-between > .item 2
.flex.row.justify-space-between > .item 3
.flex.row.justify-space-around > .item 1
.flex.row.justify-space-around > .item 2
.flex.row.justify-space-around > .item 3

Item Alignment

Add .align-items-(start/end/baseline/center/stretch) to .flex.row or .flex.col to modify item alignment.

.flex.row.align-items-start > .item 1
.flex.row.align-items-start > .item 2
.flex.row.align-items-start > .item 3
.flex.row.align-items-end > .item 1
.flex.row.align-items-end > .item 2
.flex.row.align-items-end > .item 3
.flex.row.align-items-baseline > .item 1
.flex.row.align-items-baseline > .item 2
.flex.row.align-items-baseline > .item 3
.flex.row.align-items-center > .item 1
.flex.row.align-items-center > .item 2
.flex.row.align-items-center > .item 3
.flex.row.align-items-stretch > .item 1
.flex.row.align-items-stretch > .item 2
.flex.row.align-items-stretch > .item 3

Child Classes

Customized Grow/Shrink

0 through 12 are supported for grow and shrink classes

.item.grow-5
.item.grow-2
.item.shrink-2

Ordering

0 through 12 are supported for the order classes and can be used to reorder the children

.item.order-4 1
.item.order-1 2
.item.order-3 3

Responsive Classes

All the classes mentioned above also have responsive variants of xs, sm, md, lg and xl. The default behavior is to apply to screens of all sizes. Applying a prefix to the class name will limit the class to only apply to that minimum screen size.

Prefix Minimum Widths
xs- sm- md- lg- xl-
480px 768px 992px 1200px 1480px
