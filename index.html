Flexboxy Demo

Flexboxy is a simple utility stylesheet with CSS classes representing all the common configurations of flexbox, so you don't have to write any vendor prefixing yourself.

Install

npm i --save flexboxy

Usage

If you are using webpack to include CSS with a style loader, then it's as simple as

import 'flexboxy';

Otherwise, you can download flexboxy.css and use it like any other stylesheet
