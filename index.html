Flexboxy Demo

Flexboxy is a simple utility stylesheet with CSS classes representing all the common configurations of flexbox, so you don't have to worry about vendor prefixing and you don't have to mix in flexbox with your own styles.

Install

npm i --save-dev flexboxy

Development

The build process for this project is sass to autoprefixer to cleancss. Run npm run build to make your own build. When doing development, use npm run dev to watch the source files for changes and automatically rebuild.

Usage

If you are using webpack to include CSS with a style loader, then it's as simple as

import 'flexboxy';

Otherwise, you can download flexboxy.css and use it like any other stylesheet

Documentation & Examples

Parent Classes

Direction

Row and Column directions are supported equally. The parent must explicitly describe its orientation

.flex.row > .item
.flex.row > .item
.flex.row > .item
.flex.col > .item
.flex.col > .item
.flex.col > .item

Reverse

Add the reverse class to your .flex.row or .flex.col to swap the order

.flex.row.reverse > .item 1
.flex.row.reverse > .item 2
.flex.row.reverse > .item 3
.flex.col.reverse > .item 1
.flex.col.reverse > .item 2
.flex.col.reverse > .item 3

Wrapping

Add .nowrap, .wrap or .wrap-reverse to your .flex.row or .flex.col to change wrapping behavior

.flex.row.wrap > .item 1
.flex.row.wrap > .item 2
.flex.row.wrap > .item 3
.flex.row.wrap > .item 4
.flex.col.wrap-reverse > .item 1
.flex.col.wrap-reverse > .item 2
.flex.col.wrap-reverse > .item 3
.flex.col.wrap-reverse > .item 4
.flex.col.wrap-reverse > .item 5
.flex.col.wrap-reverse > .item 6

Child Classes

Customized Grow/Shrink

0 through 12 are supported for grow and shrink classes

.item.grow-5
.item.grow-2
.item.shrink-2

Ordering

0 through 12 are supported for the order classes and can be used to reorder the children

.item.order-4 1
.item.order-1 2
.item.order-3 3
